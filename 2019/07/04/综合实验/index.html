综合实验

实验拓扑

注：如无特别说明，同一网段中，IP 地址的主机位为其设备编号，如 R3 的 g0/0 接口若在 192.168.1.0/24 网段，则其 IP 地址为 192.168.1.3/24，以此类推。

实验需求

  1. 按照图示配置P地址,R1、R2、SW1创建 Loopbacko Router–id, INTERNET 创建Loopback0 口用于模拟互联网
  2. SW2和SW3作为接入交换机,分别把PC1接入到Van10,把PC2接入到Vlan20
  3. SW1和SW2/SW3分别连接两条线缆配置链路聚合
  4. 交换机之间相连的链路配置为 Trunk,并放行相关Van
  5. sw1作为汇聚交换机,创建Van三层接口,作为Van10和Vlan20的网关
  6. R1、R2配置单区域OSPF协议,使内网P互通,使用 Loopbacko Router–id
  7. R1和R2上分别配置 EASY IP,使vlan10和vlan20的PC可以访问互联网(PC1和PC2可以Ping通 INTERNET Loopback0口)
